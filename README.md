# Sampling methods

[![MIT License](http://img.shields.io/badge/license-MIT-blue.svg?style=flat)](LICENSE)

## Summary

Sampling from a set with given weights with three algorithms, the cumulative sum, Walker's alias, and the binary tree methods.
